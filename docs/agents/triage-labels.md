# Triage Labels

This repo does not use the 5-state triage model (`needs-triage`/`needs-info`/`ready-for-agent`/`ready-for-human`/`wontfix`). Atelier is a solo project tracked on an Obsidian Kanban board with no external reporters -- cards move Backlog -> Next -> Doing -> Done manually.

Skills that reference triage roles should not apply or expect these tags on this board.
